# Генератор текстов новостей на RNN с механизмом внимания

[![Open in Google Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1VpIvRLHpS7VXSSXEsp_91hi28_JTZOX4?usp=sharing#scrollTo=44V0BEezcIQo)

## Описание проекта
Генеративная нейронная сеть на основе RNN с механизмом внимания для создания продолжений текстов на русском языке. Модель обучена на датасете новостей из русскоязычных Telegram-каналов.

## Ключевые особенности
- **Архитектура**: RNN (LSTM) с механизмом внимания и embedding-слоем
- **Данные**: 8,000 текстов из 30 русскоязычных новостных Telegram-каналов
- **Предобработка**: очистка текстов, токенизация, паддинг
- **Генерация**: функция для создания продолжений с настраиваемой температурой
- **Оценка**: метрики BLEU и анализ качества сгенерированных текстов

## Технологии
- Python 3.11+
- TensorFlow/Keras
- Hugging Face Datasets
- NLTK для оценки BLEU
- Pandas, NumPy для анализа данных

## Результаты
- **Точность модели**: ~76% на валидационной выборке
- **Качество генерации**: тексты сохраняют тематическую связанность
- **Оптимальная температура**: 0.5-0.6 для баланса связности и разнообразия
- **Вывод**: модель генерирует осмысленные продолжения, но требует доработки для улучшения связности
